Pixel Blade Chest Opener

A sophisticated automation tool with a polished GUI for opening chests in the Roblox game *Pixel Blade*.

Download Latest Version

Key Features

Polished UI

A modern dark-themed UI with an animated rainbow gradient for a premium feel.

Reliable Automation

Employs low-level `SendInput` API for hardware-like clicks that are harder for games to block.

One-Click Config

Interactively set all screen coordinates with a simple point-and-click process.

Installation & Usage

1

Download & Prepare

Download the project files (`.py` and `.bat`) and place them together in a new folder.

2

Run as Administrator

Double-click `run.bat`. A UAC prompt will appear. **You must click "Yes"** for clicks to register in-game. The script will auto-install any missing libraries.

3

Configure & Save

Use the "Set" buttons to capture coordinates for each action. Adjust timings with the sliders, then click "Save Config".

4

Start the Bot

Click the green **"START (F6)"** button or press the `F6` key to begin automation.
